### Membatalkan Revisi
Jika revisi kita blum __stagged__ ataupun __commited__, kita bisa mengembalikan menggunakan perintah `git checkout namafile`.<br>
contoh:<br>
Sebelum di ubah:
```python
print("Hello World")
```
Setelah diubah:
```python
print("Hello adnan")
```
mengembalikannya dengan cara
`git checkout namafile`
